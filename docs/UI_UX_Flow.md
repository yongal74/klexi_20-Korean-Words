  ├── 알았어요 → 다음 단어 (SRS 쉬운 경로)

- 완료 화면에서 소셜 공유 카드 자동 표시

### 2.3 퀴즈 화면 (app/(tabs)/quiz.tsx)

#### 퀴즈 상태 머신
```
[IDLE / 준비 상태]
  │ "퀴즈 시작" 탭
  ▼
[ACTIVE / 진행 중]
  ┌──────────────────────────────┐
  │  Q3 / 10          70%       │
  │  [진행 바 ========------]    │
  │                              │
  │  "사랑"의 영어 뜻은?          │
  │                              │
  │  A) friendship   ← 오답      │
  │  B) love         ← 정답 (✓)  │
  │  C) happiness                │
  │  D) dream                    │
  └──────────────────────────────┘
  정답 시: 초록 하이라이트 + 체크 애니메이션 + +15 XP
  오답 시: 빨간 쉐이크 + 정답 표시 + 오답 노트 등록
        │ 10문제 완료
        ▼
[RESULT / 결과 화면]
  ┌──────────────────────────────┐
  │  🏆 퀴즈 완료!               │
  │  8 / 10 정답 (80%)           │
  │  획득 XP: 8×15 = +120        │
  │  (9/10 이상: +50 보너스 XP)   │
  │                              │
  │  오답 단어: 2개               │
  │  [오답 복습하기]              │
  │                              │
  │  다음은?                      │
  │  [오늘 미션 확인] [홈으로]    │
  └──────────────────────────────┘
```

**UX 개선 권장:**
- 퀴즈 중 뒤로 가기 → "퀴즈를 종료할까요?" 확인 다이얼로그 추가
- 결과 화면에서 오답 단어 미리보기 (틀린 단어 + 정답 나열)
- 연속 정답 시 콤보 피드백 추가 ("3연속 정답! 🔥")

### 2.4 AI 채팅 화면 (app/ai-chat.tsx)

#### 달리(Dalli) AI 채팅 플로우
```
[진입]
  ├── 프리미엄: 무제한
  └── 무료: 오늘 사용 횟수 확인 (5회 한도)
              └── 초과 시 → 프리미엄 업그레이드 게이트 표시
  ▼
[채팅 화면]
  ┌──────────────────────────────┐
  │  🤖 달리                      │
  │  "안녕하세요! 저는 한국어 선생 │
  │  달리예요. 오늘 어떤 걸 배우고 │
  │  싶으세요?" [🔊]              │
  │                              │
  │  [내 메시지 버블]              │
  │                              │
  │  [달리 응답 버블 + 스트리밍]   │
  │  ░░░ (입력 중...)             │
  ├──────────────────────────────┤
  │  [텍스트 입력창]    [전송 →]  │
  └──────────────────────────────┘

메시지 흐름:
  사용자 입력 → POST /api/ai-chat (SSE)
             → 스트리밍 토큰 순차 렌더링
             → 완료 시 [🔊] 버튼 활성화
             → 탭 시 POST /api/ai-tts → MP3 재생
```

**UX 개선 권장:**
- 첫 진입 시 "추천 대화 주제" 버튼 3개 표시 (예: "자기소개", "날씨", "K-팝")
- 달리 응답에 핵심 한국어 단어 하이라이트 + 탭 시 단어 정보 팝업
- 대화 히스토리 저장 (현재 8개 제한 → 로컬 영속 저장으로 개선)

## 3. 핵심 사용자 여정 (User Journey)

### Journey 1: 신규 사용자 최초 경험
```
앱 설치
  → 온보딩 (4페이지: 앱 소개, TOPIK 레벨 선택, 목표 설정, 시작)
  → 게스트 로그인 (소셜 로그인 유도 but 강제 아님)
  → 홈 화면
  → "오늘의 20단어" CTA 클릭
  → 단어 학습 (첫 5단어 체험)
  → 퀴즈 유도 (학습 후 즉시)
  → 퀴즈 완료 → XP 획득 → 레벨업 애니메이션
  → 홈 복귀 → 스트릭 1일 표시
  → 푸시 알림 등록 유도 (다음날 리마인더)
```

### Journey 2: 기존 사용자 일상 루틴
```
푸시 알림 수신 ("오늘의 한국어 학습 ⏰")
  → 앱 진입 → 홈 화면
  → SRS 복습 배너 있으면 먼저 복습 (5~10분)
  → 오늘 20단어 학습 (10분)
  → 퀴즈 (5분)
  → 데일리 미션 체크
  → XP/레벨/스트릭 확인
  → 소셜 공유 (선택)
```

### Journey 3: 무료 → 프리미엄 전환
```
K-컬처 테마 진입
  → Level 2 콘텐츠 클릭
  → 프리미엄 게이트 표시
  → "프리미엄 보기" 클릭
  → 3개 플랜 비교 화면
  → 연간 플랜 선택 (가장 인기)
  → Polar 체크아웃 (외부 브라우저)
  → 결제 완료 → 웹훅 수신 → isPremium = true
  → 잠금 해제 애니메이션 + 모든 기능 접근
```

## 5. 온보딩 플로우 개선 (현재 → 개선)

### 현재 온보딩 (4페이지)
```
Page 1: 앱 소개
Page 2: 기능 나열
Page 3: 레벨 선택
Page 4: 시작하기
```

### 개선된 온보딩 (5페이지 + 첫 단어 체험)
```
Page 1: "하루 5분, 20단어로 한국어 마스터" (가치 제안 중심)
Page 2: 목표 선택 → "K-드라마 이해" / "TOPIK 시험" / "여행 준비" / "취미"
Page 3: TOPIK 레벨 자가 진단 (5개 예시 단어로 현재 레벨 추정)
Page 4: 학습 시간 알림 설정 → "매일 오후 8시" 기본값
Page 5: 첫 단어 미리보기 3개 → 즉각적인 첫 학습 체험
```

**핵심 원칙**: 온보딩 완료 전에 첫 단어를 체험하게 해서 앱의 가치를 즉시 느끼게 한다.

## 6. 접근성 (Accessibility) 체크리스트

| 항목 | 현재 | 개선 |
|------|------|------|
