> **문서 버전**: v2.0 (개선판)
> **작성일**: 2026년 3월

> **플랫폼**: iOS / Android / Web (Expo React Native)

## 1. 제품 개요 (Product Overview)

### 제품 한 줄 설명
> "하루 20단어로 6개월 안에 TOPIK 2급을 달성하는 AI 기반 한국어 학습 앱"

### 핵심 타깃
| 구분 | 내용 |
|------|------|
| 1차 타깃 | 한국어를 독학하는 해외 K-콘텐츠 팬 (18~35세, 영어권·동남아·유럽) |
| 2차 타깃 | TOPIK 시험 준비생 (중급 이상 목표, 20~40대) |
| 3차 타깃 | 한국 거주 외국인 (생활 한국어 필요, 직장인·유학생) |

### 시장 기회
- 전 세계 한국어 학습자: **약 8,500만 명** (세종학당재단, 2024)
- 연간 성장률: **18% YoY** (K-콘텐츠 붐 지속)
- 모바일 언어 학습 앱 시장 (2026): **약 $25B**
- 경쟁 앱(Duolingo, Pimsleur) 대비 TOPIK 특화·한국어 콘텐츠 깊이에서 **차별화 가능**

## 2. 문제 정의 (Problem Statement)

### 현재 학습자들이 겪는 핵심 문제

| # | 문제 | 현재 상황 | 임팩트 |
|---|------|-----------|--------|
| 1 | **비체계적 단어 학습** | Duolingo는 TOPIK 등급 체계와 무관한 단어 배열 | 시험 점수 향상 미흡 |
| 2 | **망각 곡선 미대응** | 한 번 배운 단어를 복습하는 시스템 부재 | 장기 기억 정착률 20% 미만 |
| 3 | **발음 피드백 없음** | 텍스트 기반 학습만 제공 | 실제 회화 활용 불가 |
| 4 | **한국 문화 맥락 부재** | 추상적 단어 암기만 강요 | 학습 동기 급감, 이탈률 증가 |
| 5 | **AI 튜터 부재** | 교사 없이 실전 대화 연습 불가 | 말하기 능력 미개발 |

## 3. 솔루션 및 핵심 가치 제안 (Solution & Value Proposition)

```
Klexi = TOPIK 커리큘럼 × SRS 알고리즘 × K-컬처 콘텐츠 × AI 튜터 달리
```

| 문제 | Klexi 솔루션 |
|------|-------------|
| 비체계적 학습 | TOPIK 1~6급 레벨별 7,200단어 완전 커버리지 |
| 망각 곡선 | SM-2 기반 SRS (간격 반복 학습) 자동화 |
| 발음 피드백 | AI TTS + 실제 음성 녹음 비교 (프리미엄) |
| 문화 맥락 | K-드라마·K-팝·K-푸드 등 6대 테마 3,600단어 |
| 대화 연습 | GPT-4o-mini 기반 AI 튜터 달리 (24/7 가용) |

## 4. 제품 범위 (Product Scope)

### 4.1 핵심 기능 (Core Features — Must Have)

#### F1. 일일 단어 학습 (Daily Vocabulary)
- 하루 20단어 (또는 10단어 선택 가능)
- 플래시카드 UI: 앞면(한국어) → 뒤집기 → 뒷면(영어 + 예문)
- TTS 발음 (일반 속도 / 느린 속도)
- 음절 분리 시각화 (예: 사 + 랑 + 해)
- 관련 단어 네트워크 연결

#### F2. 퀴즈 시스템 (Quiz System)
- 10문제 객관식 (4지선다)
- 오답 쉐이크 애니메이션 + 즉각 피드백
- XP 보상: 정답 +15 XP, 만점 +50 XP 추가
- 오답 노트 자동 등록

#### F3. 간격 반복 학습 (SRS Review)
- SM-2 알고리즘 기반
- 복습 예정 단어 홈화면 배너 표시
- 난이도 자기 평가 (0~5점)
- 자동 복습 스케줄 생성

#### F4. 한글 학습 (Hangeul)
- 자모 40자 완전 학습 (자음 14 + 모음 10 + 겹자음·복모음 16)
- TTS 발음 + 예시 단어
- 음절 조합 원리 설명
- 무료 사용자 전체 개방

#### F5. 문법 레퍼런스 (Grammar Reference)
- TOPIK 1~6급 108개 문법 패턴
- 15개 카테고리 (조사, 어미, 연결어 등)
- 구조 + 설명 + 예문 + 학습 팁 포함
- 카테고리 필터링

#### F6. 진도 추적 (Progress Tracking)
- 총 학습 단어, 퀴즈 횟수, 정답률
- 연속 학습 스트릭 (현재 / 최고 기록)
- 일별 학습 히스토리 캘린더

#### F7. 게임화 시스템 (Gamification)
- XP 적립 (8종 액션: 단어학습, 퀴즈, 복습 등)
- 레벨 1~8 (레벨업 효과 애니메이션)
- 15개 뱃지 (6개 카테고리)

### 4.2 고급 기능 (Advanced Features — Premium Only)

#### F8. K-컬처 테마 레슨 (K-Culture Themed Lessons)
- 6개 테마: K-드라마, K-팝, K-푸드, 여행, 인터넷 슬랭, 예절
- 테마별 TOPIK 레벨 단어 100개 × 6레벨 = 3,600단어
- **무료**: 1레벨만 / **프리미엄**: 전체

#### F9. 단어 네트워크 (Word Network)
- 형태·의미·유의어·반의어 등 10개 관계 유형 시각화
- **무료**: 첫 5개 카테고리 / **프리미엄**: 전체

#### F10. 문장 연습 (Sentence Practice)
- 빈칸 채우기 모드
- 단어 순서 배열 모드
- TTS 전문장 발음
- **프리미엄 전용**

#### F11. 발음 연습 (Pronunciation Practice)
- 3단계: 듣기 → 녹음 → 비교
- 원어민 발음 vs 사용자 녹음 파형 비교
- **프리미엄 전용**

#### F12. AI 튜터 달리 (AI Chat with Dalli)
- GPT-4o-mini 기반 한국어 회화 튜터
- SSE 스트리밍 응답
- OpenAI TTS (nova 목소리) 발음 재생
- **무료**: 하루 5회 / **프리미엄**: 무제한

#### F13. 사용자 커스텀 단어 (Custom Vocabulary)
- 직접 단어 추가
- AI 예문 자동 생성
- 복습/퀴즈에 통합

### 4.3 현재 미구현 / 개선 필요 기능 (Gap Analysis)

| 기능 | 현재 상태 | 권장 조치 |
|------|-----------|-----------|
| 소셜 로그인 (Google, Apple, Kakao) | UI만 있음 (미동작) | 우선 Google OAuth 구현 |
